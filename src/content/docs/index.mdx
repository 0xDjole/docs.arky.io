## Quick Start

```typescript
import { ArkySDK } from 'arky-sdk';

const sdk = new ArkySDK({
  apiUrl: 'https://api.arky.io',
  businessId: 'your_business_id',
  apiKey: 'your_api_key',
});

// Fetch products
const products = await sdk.eshop.getProducts({ limit: 10 });

// Create a reservation
const reservation = await sdk.reservation.createReservation({
  serviceId: 'service_123',
  providerId: 'provider_456',
  startTime: 1698765432,
  endTime: 1698769032,
});
```

## Architecture

Arky is built with modern, scalable technologies:

- **Backend**: Rust + Axum
- **Database**: TiKV (distributed transactions)
- **Search**: Qdrant (vector search)
- **Cache**: Redis
- **Analytics**: ClickHouse
- **Storage**: S3-compatible
